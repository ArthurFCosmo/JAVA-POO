## 💠Diagrama de classes

A UML é a linguagem de modelagem unificada, e nela, temos um protocolo para declaração de classes, facilitando a definição de uma classe.

Teremos:

- O nome da classe como um "título"
- Um bloco com os atributos
- Um bloco com os métodos

Isso tudo facilita a representatividade das classes objetos com seus atributos e métodos.

Começa com letra maiúscula = Classe
Todas as letras minúsculas e termina com parênteses = Método
Todas as letras minúsculas = Atributo
